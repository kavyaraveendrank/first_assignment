  3. Frontend:

    frontend is what your users see and includes visual elements like buttons, checkboxes, graphics, and text messages.
    It allows your users to interact with your application.A Front-End Developer is someone who creates web applications.
    The difference between Front-End and Back-End is that Front-End refers to how a web page looks,
    while back-end refers to how it works. The basic languages for Front-End Development are HTML, CSS, and JavaScript.

    4. Backend:

    The backend is the data and infrastructure that make your application work. It stores and processes application data for your users.
    You can think of Front-End as client-side and Back-End as server-side.Back-end developers ensure the website performs correctly,
    focusing on databases, back-end logic, application programming interface (APIs), architecture, and servers.
